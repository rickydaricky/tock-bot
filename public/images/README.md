# Icon Files

This directory should contain the following icon files:

- `icon16.png` - 16x16 pixel icon
- `icon48.png` - 48x48 pixel icon  
- `icon128.png` - 128x128 pixel icon

You can generate these icons from the `icon.svg` file using image editing software or the script in `scripts/generate-icons.js` (requires additional setup).

For development purposes, you can create simple placeholder PNGs or use the SVG directly in modern browsers.
